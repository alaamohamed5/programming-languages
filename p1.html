A programming language is a vocabulary and set of grammatical rules for instruction to perform specific tasks on a computer or computing device. All languages (including assembly languages) at least assign a type to a value before it is operated on, as operations are not well-defined without data interpretation. Most languages also assign types to values which are stored simply in memory.
One set of languages which is not assembly languages: values in memory (including registers) are only bytes and the programmer must implicitly keep track of their interpretation. Statically typed languages contain explicit statements limiting values types a to which a variable may be bound. The C++ and Java languages are statically typed.
Dynamically typed languages like Scheme and Python allow a variable to be bound to any value.
Some languages, such as ML, are typed dynamically but the interpreter uses type inference to assign static types autonomously where possible.
